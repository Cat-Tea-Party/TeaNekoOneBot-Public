# 猫猫帝国

## 介绍
以猫咪作为底层打工猫建设猫猫帝国的放置挂机游戏。

玩家可以通过建造不同的设施来打造猫猫帝国，招募更多的猫猫来帝国打工。
设施可以定时来产生不同的材料和物资。
材料可以用来建造、升级建筑措施，物资可以保证猫猫们的基本生存以及提高工作效率。
